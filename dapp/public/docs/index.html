Getting Started

1. Create Your Wallet

Click "Connect Wallet" and choose "Create New Wallet". You'll receive:

  • 24-word recovery phrase - Write it down and store securely!
  • EVM Address (0x...) - MetaMask compatible, for Ethereum ecosystem
  • Quantum Address (neo1...) - Post-quantum safe for future security

2. Get Test Tokens

Use our faucet to get test tokens:

curl -X POST https://faucet.neonet.ai/request \
  -H "Content-Type: application/json" \
  -d '{"address": "YOUR_ADDRESS"}'

3. Make Your First Transaction

Go to the "Send" tab, select a token, enter recipient address and amount. Click "Send Transaction".

GPU Mining

Quick Start

# Install NeoNet Miner
curl -sSL https://neonet.ai/install | bash

# Configure your wallet
neonet-miner config --wallet YOUR_ADDRESS

# Start mining
neonet-miner start --gpu auto

API Reference

Base URL

Authentication

Authorization: Bearer YOUR_API_KEY

Endpoints

MethodEndpointDescription
GET/healthHealth check
GET/network/statsNetwork statistics
POST/transactionsSubmit transaction

POST/tasksSubmit AI task

Example: Submit Transaction

  -H "Content-Type: application/json" \
